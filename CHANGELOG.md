# Changelog

All notable changes to this project are documented here.

## [0.1.0] - Unreleased

### Added
- Initial release
- `PDF4.dev` node with two resources: **PDF** and **Template**
- PDF operations: **Render From Template**, **Render From HTML**
- Template operations: **List**, **Get**
- `Pdf4Api` credential (Bearer token, base URL override)
- Binary + base64 output modes
- Format overrides (preset, custom width/height)
- Light + dark SVG icons
